# Features:

The website has three apps and one welcome page 
- Open Welcome in the header leads to a welcome page
- The three desktop apps that lead to the other apps
- Multiple draggable windows
- Z-ordering by last active window
- Add and delete notes
- Find a virus in the internet app
- Watch the stopwatch turn to hour format
